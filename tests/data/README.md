# Benchmark reference data

Generated by the Mathematica jleGroup occLightCurves `olcOneLimb2` model (EY92 family):
one-limb light curves, instantaneous sampling, λ = 0.7 µm. Geometry (all cases, see each
`parameters.csv`): N₂ atmosphere, observer at 30 AU, closest approach b = 900 km,
shadow velocity 25 km/s, body mass 2.1398 × 10²² kg — which is
`pdMass["Triton", 1]` (Anderson et al. 1992): the test body is Triton, by mass.

Atmosphere tables are machine-exact hydrostatic profiles with power-law temperature
T ∝ r^b, radii 1200–2500 km at 1 km steps, in Mathematica float format (`1.0865*10^22`):

| case | b | T range |
|---|---|---|
| iso-clear | 0 | 114.52 K (isothermal) |
| shallow-clear | −0.5 | 118.8 → 82.3 K |
| steep-clear | −4.5 | 166.7 → 6.1 K |

The steep-clear *reference* carries a known ~1 × 10⁻³ deep-region error from the
generator's first-order-in-1/λ EY92 truncation (see `tests/test_benchmarks.py`).

The generator notebook lives in the maintainer's research materials
(`generate-test-cases.nb`), not in this repository.
