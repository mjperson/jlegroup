# EY92 paper benchmark (Elliot & Young 1992, AJ 103, 991)

Digitized from the published paper — NOT generated by the Mathematica
jleGroup. This makes it an oracle independent of the whole
Mathematica/Python lineage: it validates the EY92 *method* against the
authors' own 1992 Mathematica computation (19-digit precision, ~7
printed significant digits).

- `table3a.csv`, `table3b.csv` — EY92 Tables 3(a) (T exponent b = 0)
  and 3(b) (b = −0.6): the benchmark light-curve tables, digitized from
  a 400-dpi rasterization of journal page 1005. One row per
  (time, series-truncation-order); orders are 0, 1, 4. Column notes:
  `nu_1e9` is refractivity /1e-9; `kappa_1e8_cm` is the haze absorption
  coefficient /1e-8 cm⁻¹ (the journal's column header prints garbled
  units "(μm² cm⁻³)"); `s_i` is the binned signal in ADU per 0.2 s.
  Digitization is verified by internal over-determination in
  `tests/test_EY92.py` (φ = e^(−τ)·φ_ref, s_i = Δt(s_b + s_*φ), ρ(t)
  geometry, λ_g↔r↔ν column relations).
- `p1004_derived.csv` — the paper's derived atmospheric parameters
  (p. 1004) for the same benchmark, by truncation order; the target of
  the data→atmospheric conversion test. **Known misprint:** the b = 0,
  order-4 `lambda_g0` (20.95567) is inconsistent with the paper's own
  Table 3(a), which implies ≈20.9567 (= what the conversion in
  `jlegroup.EY92` reproduces); the test tolerances document this.

Benchmark inputs (paper Table 2): the EY92 "data parameter set" —
s_b = 630, s_f = 3340 (counts/s), t_im = 53.9 s, t_em = 139.9 s,
T_Hiso = 4.7 s, a = 0, T_h,1 = 4.2 s, T_h,2 = 7.0 s, T_Hτ2 = 2.3 s,
T_min = 46.9 s, v = 18.5 km/s, Δt = 0.2 s, D = 28.89675 AU, r0 = 1250 km.

Errata established for the published paper during this validation
(consistent with the "Chamberlain 1996" b-series error noted in the
Mathematica jleGroup usage text): Eq. (A5) spurious 1/2 on dA/dδ;
Eq. (A2) δ⁴b³ denominator 8192 → 32768; Eq. (A6) δ⁴b² numerator
10555 → 1055; and the p. 1004 entry above. See the `jlegroup/EY92.py`
module docstring.
